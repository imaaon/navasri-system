📖 คู่มือตั้งค่า Line Notification
💡 ใช้ n8n (แนะนำ — ฟรี self-host หรือ cloud trial) หรือ Make.com (ฟรี 1,000 operations/เดือน) เป็นตัวกลางเชื่อม Webhook → Line
A สร้าง Line Official Account & Messaging API
1
ไปที่ developers.line.biz → Login → Create Provider
2
Create Channel → เลือก Messaging API → กรอกข้อมูล OA
3
ไปที่แท็บ Messaging API → ก๊อป Channel Access Token (Long-lived) เก็บไว้
4
เพิ่ม Bot เข้า Line Group ที่ต้องการแจ้งเตือน จากนั้นส่งข้อความใดก็ได้ใน Group
5
หา Group ID: ใช้ Line Bot API เรียก GET /v2/bot/group/{groupId}/summary หรือดูจาก webhook event แรก
B สร้าง Workflow ใน n8n
1
ไปที่ n8n.io → Sign Up (ฟรี Cloud trial) หรือ Self-host
2
New Workflow → Add Node → เลือก Webhook
3
ตั้งค่า Webhook: Method = POST, Authentication = None → Copy Webhook URL
4
เพิ่ม Node → HTTP Request → ตั้งค่าดังนี้:
URL: https://api.line.me/v2/bot/message/push
Method: POST
Header: Authorization: Bearer {Channel Access Token}
Header: Content-Type: application/json
Body (JSON):
{
  "to": "{Group ID}",
  "messages": [{"type": "text", "text": "{{ $json.message }}"}]
}
5
บันทึก Workflow → Activate → กลับมากรอก Webhook URL ในระบบนี้
C ทางเลือก: Make.com
1
make.com → Create Scenario → Add Module → Webhooks → Custom Webhook
2
Copy Webhook URL → ต่อ Module HTTP → Make a Request → ยิงไป Line API เหมือน n8n
3
หรือใช้ Module Line → Send a Message โดยตรง (ต้อง Connect Line account)
